# GreenLeaf Token (GLT) - MiniBank

## 项目简介

GreenLeaf Token (GLT) 是一个基于ERC20标准的代币合约，具有独特的经济模型和功能特性。该合约实现了代币发行、转账限制、账户冻结、手续费机制等功能，旨在构建一个安全可控的代币生态系统。

## 功能特性

### 1. 基础ERC20功能
- 代币名称：GreenLeaf Token
- 代币符号：GLT
- 小数位数：18位
- 初始供应量：1亿枚（100,000,000 GLT）

### 2. 管理员功能
- **铸币功能**：合约所有者可以向指定地址增发代币
- **账户冻结/解冻**：可以冻结或解冻特定账户，被冻结账户无法进行转账操作
- **白名单管理**：可设置不受转账限制的白名单地址
- **所有权转移**：支持合约所有权的转移

### 3. 转账限制机制
- **早期阶段限制**：合约部署后180天内，普通用户单日转账额度不超过其持有量的5%
- **白名单豁免**：白名单地址不受转账限制约束
- **自动解除**：180天后自动解除转账限制

### 4. 手续费机制
- **税率设置**：默认税率为2%（200 BPS）
- **收款地址**：可设置手续费收款地址
- **动态调整**：合约所有者可调整税率参数

### 5. 安全特性
- **防抛售机制**：通过每日转账限制防止大额抛售
- **账户控制**：支持对违规账户进行冻结
- **权限管理**：关键功能仅限合约所有者执行

## 技术架构

- **开发框架**：Hardhat
- **编程语言**：Solidity ^0.8.2
- **标准库**：OpenZeppelin Contracts
- **测试框架**：Mocha + Chai + ethers.js

## 安装与使用

### 环境要求
- Node.js >= 16.x
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 编译合约
```bash
npx hardhat compile
```

### 运行测试
```bash
npx hardhat test
```

### 部署合约
```bash
npx hardhat run scripts/deploy.ts --network <network-name>
```

## 测试说明

项目包含完整的测试套件，覆盖以下场景：

### 1. 初始分配测试
- 验证部署者获得全部初始供应量

### 2. 手续费机制测试
- 验证转账时正确扣除2%手续费
- 验证手续费正确转入指定收款地址

### 3. 账户冻结测试
- 验证冻结账户无法进行转账操作
- 验证错误提示信息正确性

### 4. 转账限制测试
- 验证180天内的5%日转账限制
- 验证超额转账会被拒绝
- 验证限额内转账正常工作

### 5. 时间限制解除测试
- 验证180天后转账限制自动解除
- 验证时间推进后的正常转账功能
